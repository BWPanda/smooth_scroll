
-- SUMMARY --

This module is meant for One pager websites, and sitebuilders who have no
knowledge of js or just want to build their site up quickly and use this effect.

By using the Smooth Scroll module the effect is applied to all anchor links
on your Drupal setup. http://css-tricks.com/snippets/jquery/smooth-scrolling/

You can set the configurations for the module in config/user-interface/smooth-scrolling.

For a full description of the module, visit the project page
  http://drupal.org/project/smooth_scrolling

To submit bug reports and feature suggestions, or to track changes
  http://drupal.org/project/issues/smooth_scrolling


-- REQUIREMENTS --

None.


-- INSTALLATION --

 Install as usual.


-- CONFIGURATION --

 Configuration is set in
  Administer > Settings > Smooth Scrolling

 Customize the menu settings in Administer  Site configuration
  Administration menu.

 To prevent administrative menu items from appearing twice, you may hide the
  Navigation menu block, or move the Administer menu items into a separate
  menu.
